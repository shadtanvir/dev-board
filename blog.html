Question-1: What are the different ways to select an element in the DOM?

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelectorAll()
  • querySelector()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • innerHTML: It is used to access the html content inside an element including html tags
  • innerText: It is used to get or set the visible text content of an element
  • textContent: It helps us to access the raw text content(doesn't care about the text styles) inside an element including text from hidden elements unlinke innerText

Question-3: What is event delegation in the DOM?

Event delegation is a technique which helps to handle an event by the child elements which was added for the parent element. It brings more efficiency to our program.

Question-4: What is event bubbling in the DOM?

Event bubbling is a DOM mechanism where an event triggered on a child element propagates upward through its ancestors in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

  • Create: document.createElement('element')
  • Add: document.parentElement.appendChild(element)
  • Remove: parentElement.removeChild(element)
